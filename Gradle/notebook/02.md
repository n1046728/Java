Groovy
=====
1. 用於java虛擬機的一種敏捷的動態語言，他是一種成熟的物件導向程式語言，亦可用做純粹的腳本語言。
2. 與java比較
> * Groovy兼容java語法  
> * 分號可省略  
> * class、method預設為public
> * 編譯器給屬性自動添加getter/setter方法  
> * 最後一個表達式的值會被作為返回值
> * ==等同於equals()，不會有NullPointerExceptions
3. Groovy特性
> * assert語句
> * 可省略括號
> * 集合api
> * 可選類型定義
> * 字符串
> * 閉包  

範例1.java vs groovy
----
* java
```java

	private int minor;

		this.major = major;
```  
